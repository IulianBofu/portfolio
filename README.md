# Cloud & Cybersecurity Portfolio

This repository contains a curated set of labs, scripts and notes showcasing my work in cybersecurity, cloud infrastructure, servers and automation.

## Structure

- `cybersecurity/` – log analysis, IDS/IPS-style experiments, incident response notes.
- `cloud-and-servers/` – Azure labs, Linux/Windows server hardening and GPO baselines.
- `automation-scripts/` – Python and shell scripts for backups, monitoring and anomaly detection.
- `docs/` – portfolio overview and documentation.
- `community-and-learning/` – volunteering, running goals and learning roadmap.
